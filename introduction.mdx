## What is SMyEG?

SMyEG (Sistema de Monitoreo y Evaluación Geoespacial) is a comprehensive full-stack platform designed for managing forest and land patrimony with advanced geospatial capabilities. Built for organizations that need to track, monitor, and evaluate land assets across multiple hierarchical levels.

## Technology Stack

SMyEG is built with modern web technologies:

- **Frontend**: Next.js 16, React 19, TypeScript
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL with PostGIS extensions
- **Maps**: Leaflet and React Leaflet for geospatial visualization
- **Authentication**: NextAuth v5
- **UI**: Tailwind CSS, Radix UI, shadcn/ui components
